Gene count software

This software uses gene count data across species, i.e. the number of gene copies in each species for each family, as in the software CAFE of Hahn et al., Genome Research (2005). The originality of this software is in the fact that it incorporates Whole Genome Duplication (WGD) events. It estimates the birth rate, the death rate and the probability of loss at the WGD.
